# Summer 60 Habit Tracker

A simple personal habit tracker for the Summer 60 challenge.

This tracker helps you follow daily habits, weekly progress, and overall consistency across 60 days.

## Live Website

Add your GitHub Pages link here after deployment:

https://SACHIN-PARAJULI.github.io/Habit-tracker/summer_60.html

## Features

1. Daily habit tracking for 60 days

2. Weekly habit view

3. Progress percentage for each week

4. Reports dashboard

5. Habit completion ranking

6. Perfect day and streak tracking

7. Data saved directly in the browser

## Habits Included

1. Circadian Anchor: First Priority

2. Early Surgical Strike: 120 min

3. Cave Mode: 90 min

4. Finisher Mode: 60 min

5. 6 Hour Work / Day

6. Workout

7. Meditation

8. Walk When Restless / Alone / Tired

9. Say Less in One Conversation

10. Hydrate

11. 5 Min Reflection

## How It Works

Open the tracker in a browser and mark each habit as completed for the day.

The progress is saved in the same browser using local storage. This means your data stays on the device and browser where you use the tracker.

## Important Note

This tracker does not use a database or login system.

If you open it from another phone, laptop, or browser, your checked habits will not automatically sync.
